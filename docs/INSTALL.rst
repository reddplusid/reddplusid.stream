reddplusid.stream Installation
------------------------------

To install reddplusid.stream using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``reddplusid.stream`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        reddplusid.stream

* Re-run buildout, e.g. with:

    $ ./bin/buildout
